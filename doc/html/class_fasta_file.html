Detailed Description

class that describes file of FASTA format

This is wrapper around std::ifstream for easier work with .fasta files. It enables simple iterating through records in opened file.

See also
http://www.bioinformatics.nl/tools/crab_fasta.html
http://www.cplusplus.com/reference/fstream/ifstream/
