FastaRecord::MinimizersTable FastaRecord::getMinimizers ( short  k,
short  w 
) const

Computes and returns vector of k-mer minimizers from record's attribute sequence_.

Computes and returns vector of k-mer minimizers from record's attribute sequence_. Minimizers are uniquely chosen representative k-mers from a group of adjacent k-mers. Thus, each minimizer represents one sequence's window. By using minimizers, memory usage can be reduced significantly.

Having w <= k guarantees that no gaps appear between adjacent minimizers. [2]

Parameters
ksize of a k-mer
wnumber of consecutive adjacent k-mers in a window.
Returns
unordered_map of minimizers to positions where they occur
