Aligns whole read to reference.

Finds mapping of whole read on reference and aligns it with Smith-Waterman algorithm. Whole read sequence and its reference mapping sequence are used. Starting index on reference is found from first seed in given band, which is then shifted for corresponding distance of starting index on the read sequence.

Example: read: ____FFFF______LLL_____ used: ++++++++++++++++++++++

  • where F marks first minimizer and L marks last minimizer and where - marks unused portion of read and + marks used portio
